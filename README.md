# Glyph Weaver

A tool for crafting and manipulating symbolic glyph systems, runic alphabets, and custom character sets.

## Overview

Glyph Weaver provides a framework for designing, transforming, and rendering symbolic glyph systems. Whether you're building a fictional runic alphabet for a game, designing a custom icon set, or exploring the geometry of symbolic writing systems, Glyph Weaver gives you the primitives to define, compose, and export glyphs programmatically.

## Getting Started

```bash
pnpm install
pnpm run-all-checks
```
